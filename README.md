# Supplier NER — извлечение поставщика из закупочной документации

Дообучение модели ruBERT-base-cased для автоматического извлечения наименования поставщика из протоколов аукционов.

## Результаты

| Метрика | Значение |
|---------|----------|
| Precision | 0.7244 |
| Recall | 0.9209 |
| F1-score | 0.8109 |

## Структура проекта

- `ner_utils.py` — базовые утилиты (токенизация, поиск)
- `fuzzy_search.py` — нечёткий поиск (fuzzysearch)
- `synonyms.py` — словарь синонимов организационно-правовых форм
- `filter_example.py` — фильтрация шумных примеров
- `bio_alignment.py` — выравнивание токенов и BIO-разметка
- `filter_data.py` — обёртка для фильтрации датасета
- `prepare.py` — подготовка данных для обучения
- `simple_train.py` — дообучение модели
- `predict.py` — тестирование модели
- `config.py` — конфигурация (пути, гиперпараметры)

## Установка

```bash
pip install -r requirements.txt
```
